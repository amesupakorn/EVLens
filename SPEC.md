# Project Overview: EVLens
**Goal:** เว็บไซต์เครื่องมือช่วยตัดสินใจสำหรับผู้ที่ต้องการเปลี่ยนจากรถน้ำมันมาเป็นรถไฟฟ้า (EV) ในประเทศไทย

---

## 1. Core Problem Statement
ผู้ใช้ส่วนใหญ่ที่ต้องการเปลี่ยนมาใช้ EV มักมีความกังวล (Anxiety) ใน 3 เรื่องหลัก:
1. **Range Anxiety:** กลัววิ่งไม่ถึงที่หมาย หรือไม่รู้ว่าต้องชาร์จบ่อยแค่ไหน
2. **Value Uncertainty:** ไม่แน่ใจว่าประหยัดกว่าน้ำมันจริงไหม และคุ้มค่าในระยะยาวหรือไม่
3. **Accessibility:** ไม่รู้ว่ารุ่นที่สนใจ ชาร์จที่ไหนได้บ้าง

## 2. Main Functional Requirements (ฟีเจอร์หลัก)

### A. EV Range Calculator
- **Inputs:** ระยะทางขับขี่ต่อวัน (กม.), รุ่นรถที่สนใจ
- **Logic:** คำนวณจากแบตเตอรี่รุ่นนั้นๆ (kWh) เทียบกับอัตราสิ้นเปลือง เพื่อสรุปว่า "ชาร์จ 1 ครั้ง ขับได้กี่วัน" 
- **Output:** จำนวนวันที่ต้องชาร์จต่อสัปดาห์ และเปอร์เซ็นต์แบตเตอรี่ที่เหลือเมื่อกลับถึงบ้าน

### B. Cost & Savings Analytics
- **Inputs:** ราคาน้ำมันปัจจุบัน (บาท/ลิตร), อัตราสิ้นเปลืองรถคันเดิม, ค่าไฟ (เลือกได้ระหว่าง Standard หรือ TOU)
- **Output:** ตารางเปรียบเทียบค่าใช้จ่ายรายเดือน/รายปี และจุดคุ้มทุน (Break-even point)

### C. Smart EV Directory
- ระบบ Database รถ EV ในไทย (แบรนด์, รุ่น, ราคา, ขนาดแบตเตอรี่, ระยะทางวิ่งสูงสุด)
- ระบบ Filter: กรองตามงบประมาณ (Price Range) และประเภทรถ (Sedan, SUV, Hatchback)

### D. Charger Locator Integration
- เชื่อมต่อ API หรือ Link ไปยังตำแหน่งสถานีชาร์จ (Google Maps) โดยเน้นค่ายหลักในไทย เช่น EA Anywhere, PEA Volta, PTT EV Station

## 3. Non-Functional Requirements
- **Performance:** หน้าเว็บต้องโหลดเร็ว (Lighthouse score > 90) โดยเฉพาะส่วนของ Calculator
- **Responsiveness:** รองรับการใช้งานบน Mobile 100% (เพราะคนมักเปิดเช็กตอนอยู่ที่โชว์รูมหรือบนรถ)
- **Data Source:** ข้อมูลราคารถและสเปกต้องอัปเดตตามตลาดไทยปี 2026

## 4. User Flow Summary
1. **Landing:** รับทราบ Value Proposition (ประหยัดแค่ไหน/ไปได้ไกลเท่าไหร่)
2. **Input Data:** กรอกพฤติกรรมการขับขี่
3. **Result:** เห็นรุ่นรถที่แนะนำ + ตัวเลขความประหยัด
4. **Action:** ดูรายละเอียดรถแต่ละรุ่น หรือหาจุดชาร์จใกล้ตัว
